📱 功能演示

1.![登陆界面]（https://s41.ax1x.com/2026/05/02/pe7Ed0g.png）
用户打开软件后，进入登录页面。该页面提供手机号验证码登录方式。

2.![首页]（https://s41.ax1x.com/2026/05/02/pe7E71x.png）
用户登陆后并且进入主页面后，可在此页面进行模块选择

3.![浏览活动]（https://s41.ax1x.com/2026/05/02/pe7EHc6.png）
进入活动页面后会显示多个活动，并且显示活动名称，简介，地点，日期，人数，活动类型，难度以及剩余名额

4.![创建活动]（https://s41.ax1x.com/2026/05/02/pe7EbjK.png）
进入活动创建页面后，可以看到如上图所示的各种活动创建时的信息，例如：活动标题，活动描述，活动类型，难度等级，活动地点，开始以及结束时间，最大参与人数，模板类型，还可以在此处上传活动的封面图片

5.![浏览路线]（https://s41.ax1x.com/2026/05/02/pe7ELnO.png）
进入查看路线的界面，在此界面，用户可以查看其他玩家分享的路线

6.![分享路线]（https://s41.ax1x.com/2026/05/02/pe7VkDS.png）
进入路线分享界面，进行自己的路线分享，用户需要在此处填写路线标题，路线描述，地点，距离，时长以及难度的设置

7.![找搭子界面]（https://s41.ax1x.com/2026/05/02/pe7VMvV.png）
用户可通过进入找搭子界面来进行同伴的寻找或者发布寻找同伴的告示

8.![发布找搭子需求界面]（https://s41.ax1x.com/2026/05/02/pe7exN4.png）
用户可通过单击发布需求按钮，来发布自己所需的搭子的需求，来找到志同道合的运动伙伴

9.![个人中心界面]（https://s3.bmp.ovh/2026/05/02/V66uRlxu.png）
用户可通过单击右上角如图红色标示框所标示的头像，来查看各种信息
